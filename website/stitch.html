cowsay-rs

A friendly cowsay for the Rust era.

cowsay-rs is a modern, fast, and fun rewrite of the classic cowsay command-line program, built with Rust and WebAssembly.

Try cowsay-rs in your browser!

Get Your Own Cow.

Install with cargo

cargo install cowsay-rs

Use in your Web Project

npm install @cowsay-rs/web

About This Project

Hello! I'm Franco, a software developer passionate about open-source and learning new technologies. I created cowsay-rs as a fun weekend project to dive deeper into the Rust programming language and its ecosystem. It started as a simple command-line tool and quickly evolved into a fun experiment with WebAssembly, allowing it to run directly in the browser.

What's the use of a Rust cow?

  • Learning Rust in a fun, hands-on way.
  • Exploring the power of WebAssembly (WASM).
  • Giving back to the developer community with a useful (and amusing) tool.
